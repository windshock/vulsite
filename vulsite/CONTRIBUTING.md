# Contributing

## Collaboration Rules (Minimal)
1) Branch: feature/<name>-<topic>
2) PR must include: What / Why / How to test
3) Merge requires 1 approval
4) Create an issue for bigger changes
5) Keep commits small and focused

## 협업 규칙 (간단)
1) 브랜치: feature/<name>-<topic>
2) PR에 반드시 포함: 무엇 / 왜 / 테스트 방법
3) 머지 전 1명 승인 필요
4) 큰 변경은 이슈 생성
5) 커밋은 작고 집중된 단위로
